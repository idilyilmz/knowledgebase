## What is Cowork?

Cowork is Anthropic's tool for knowledge work, built on the same architecture as Claude Code. While chat is great for quick Q&A and back-and-forth refinement, Cowork shifts the interaction from *conversation* to *delegation*: you describe an outcome, and Claude plans the steps, executes them, and delivers a finished file.

Three things make this work:
- **Plan** — For multi-step tasks, Claude shows its approach before starting so you can review and approve.
- **Execute** — Work runs in an isolated environment on your computer and can keep going while you step away.
- **Connect** — Cowork reaches into the tools where your work already lives (email, drives, etc.), so context flows in without copy-paste.

Its six core capabilities are connectors, file operations, plugins (role-specific expertise bundles), scheduled tasks, subagents (parallel work), and local computation.

**When to use which:** Reach for Cowork when you want a real output file, when the task touches your drive or connected tools, when there are many steps or items, or when you'd like work to run in the background. Stick with chat when you just want a quick answer, a draft to refine yourself, or to think something through turn by turn.

The lesson ends by asking you to think of one task from your week that fits the Cowork shape — you'll use it in Lesson 3.

## Getting set up

This lesson walks you through setting up Cowork so you're ready to run your first task.

**What you need:** Cowork runs in the Claude Desktop app and is available on paid plans. To open it, launch Claude Desktop and use the mode selector at the top to switch from Chat to Cowork.

**The key setup step — point Claude at a folder:** Click *Work in a folder* and select a directory on your computer. This is what makes Cowork different from chat: Claude can read every file in that folder and save finished work back there directly, with no uploading or downloading. It asks permission before making changes.

**Optional but powerful additions:**
- **Connectors** (set up in the Customize area) link Cowork to services like Slack, Gmail, Google Drive, and Calendar. Once connected, you can reference them naturally in prompts ("check what the team said in Slack") instead of pasting content in.
- **Claude in Chrome** extends Cowork's reach to web pages that don't have a connector — internal dashboards, vendor portals, or anything behind a login. You install it from the Customize area and grant per-site permissions.

**The bottom line:** a folder is enough to get started. Connectors and the Chrome extension are there when you need them, and projects (which add persistent instructions and memory to a folder) come in Lesson 4.

Your practice task: open Claude Desktop, switch to Cowork, and point it at a folder you'd like to work with. The next lesson covers the core loop of actually running a task.
